# @illusions-lab/mdi-to-hast

Maps MDI mdast nodes to hast, following the HTML mapping defined in SYNTAX.md. Shared foundation for @illusions-lab/mdi-to-html, @illusions-lab/mdi-to-pdf, and @illusions-lab/mdi-to-epub.

Part of the [MDI](https://github.com/illusions-lab/MDI) monorepo. See the root README for the full package architecture.

Documentation: https://mdi.illusions.app/
